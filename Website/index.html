The Project

The core of our project is our deep learning model using OpenCV. Our deep learning model is able to identify the difference from a vacant parking spot, and a filled one. Our model edits the orignal photo/video so that it shows which spots are open by highlighting them green. Scroll below to see the model in action.

Our smart parking system will send a user a notifcation via email of when a parking spot opens in their desired parking spot. This feature integrates a Radar.io location finder and a database of parking spots. A user can enter their name, email, and parking lot, and when a spot opens they will be notified. The code for this function is ready, but the feature is more proof of concept becuase we do not have much acces to live footage , however if given this footage, we would easily be able to implement our existing code to get it fully functioning with the feed.

Frequently Asked Questions

  • How does the model work?

    Our model first breaks down the individual parking spots of a parking lot. From here we distinguish filled spots from non filled spots. After we are able to recognize the difference, we highlight the open spots in green, editing the oringal photo or video.

  • What locations are you currently working in?

    Currently, our code only works for parking lots in a large majority of the Bay Area. However, as we progress we will be able to collect more data from other locations and analyze their parking lots.

  • What is the use of the app?

    You will be able to find and be notified of open parking spaces in the parking lot you have chosen as soon as they are available. Thus, saving you time and money, because you will not waste gas looking for a spot, and our database only contains free parking spots so you don't have to pay for any pesky parking fees.
